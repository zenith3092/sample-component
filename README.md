# Sample Component

1. Toggle
2. Polygon Drawing Tool

## Environment Variables

-   `VITE_RTSP_URL`
-   `VITE_WEB_SERVER_URL`

## Runtime Requirements

-   `Node.js 18.17.0^`
-   `npm 9.6.7^`

## Installation

```bash
npm install
```

## Start

```bash
npm run dev
```

If you want to expose your local ip:

```bash
npm run dev -- --host
```
